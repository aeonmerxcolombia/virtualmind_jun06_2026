Estructura del Backend

Ubicación: /home/ubuntu/backend

├── app/
│   ├── auth/          # Autenticación, JWT, hashing
│   ├── database/      # Conexión a BD
│   ├── models/        # Modelos SQLAlchemy
│   ├── routes/        # Endpoints API
│   ├── schemas/       # Validación Pydantic
│   └── services/       # Lógica de negocio
├── mcp_server/        # MCP tools (8001)
├── mcp_ollama/        # Ollama AI (8002)
├── static/
├── uploads/
└── env/              # Virtual environment

Comandos de Mantenimiento

Reiniciar Backend

pkill -f uvicorn && cd /home/ubuntu/backend && source env/bin/activate && nohup uvicorn app.main:app --host 0.0.0.0 --port 8000 --ssl-keyfile .../privkey.pem --ssl-certfile .../fullchain.pem > /tmp/backend.log 2>&1 &

Ver Logs

tail -f /tmp/backend.log

Ver Servicios

ps aux | grep -E "uvicorn|apache2|mysql"

Acceso MySQL

mysql -u root -pcatedra2025 virtualmind_db
